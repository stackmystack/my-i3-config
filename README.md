# My i3 config

## Getting Started

### Dependencies

- i3wm
- i3lock
- clipit
- blueman-applet
- nm-applet
- nitrogen
- pactl
- rofi

### Installation

~~~ bash
$ mv ~/.config/i3 /tmp/i3-old
$ cd ~/.config
$ git clone https://github.com/tommyku/my-i3-config i3
$ mv /tmp/i3-old ~/.config/i3
~~~
